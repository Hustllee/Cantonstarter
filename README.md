# CantonStarter Bar

A lightweight web widget that displays ecosystem updates and recommended accounts to follow on the Canton Network.

## Overview

CantonStarter is a simple developer tool designed to help users quickly discover what's happening in the Canton ecosystem.

The bar can be embedded on any website and provides:

- Latest ecosystem updates
- Key builders and "Chads" to follow
- Quick access to important Canton resources

## Features

- Live-style update feed
- Suggested accounts to follow
- Lightweight UI bar that sits at the bottom of a webpage
- Easy integration into any web project

## Built With

- HTML
- CSS
- JavaScript

## Future Improvements

- Real-time updates using the X API
- Dynamic Canton ecosystem news
- Builder leaderboard
- AI-powered ecosystem discovery

## Vision

The goal of CantonStarter is to make it easier for newcomers to discover the Canton ecosystem and quickly connect with the most important builders, updates, and conversations happening around Canton.
